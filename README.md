# lyrics-fetcher

Lyrics Fetcher is a simple tool which provides ability to fetch lyrics on japanese lyrics websites.

## Supported Sites
* [animap.jp](http://animap.jp)
* [evesta.jp](http://evesta.jp)
* [j-lyric.net](http://j-lyric.net)
* [jtw.zaq.ne.jp/animesong](http://jtw.zaq.ne.jp/animesong)
* [kashinavi.com](http://kashinavi.com)
* [kasi-time.com](http://kasi-time.com)
* [kget.jp](http://kget.jp)
* [utamap.com](http://utamap.com)
* [uta-net.com](http://uta-net.com)
* [utaten.com](http://utaten.com)
