# lyrics-fetcher

Lyrics Fetcher is a simple tool which provides ability to fetch lyrics on japanese lyrics websites.

## Supported Sites
* [animap.jp](animap.jp)
* [evesta.jp](evesta.jp)
* [j-lyric.net](j-lyric.net)
* [jtw.zaq.ne.jp/animesong](jtw.zaq.ne.jp/animesong)
* [kashinavi.com](kashinavi.com)
* [kasi-time.com](kasi-time.com)
* [kget.jp](kget.jp)
* [utamap.com](utamap.com)
* [uta-net.com](uta-net.com)
* [utaten.com](utaten.com)
